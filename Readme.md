## 🗓️ 6-Week Study Plan Summary

| Week | Focus | Domain |
|------|-------|--------|
| **Week 1** | Baseline assessment + Sentinel workspace, data connectors, AMA | Domain 1 |
| **Week 2** | Analytics rules, automation rules, playbooks (Logic Apps), ASR rules | Domain 1 |
| **Week 3** | Incident response — Defender for Office 365, Identity, Entra ID, Copilot for Security | Domain 2 |
| **Week 4** | Defender for Cloud, Cloud Apps (CASB), Purview Audit — mid-point mock exam | Domain 2 |
| **Week 5** | KQL deep dive, Advanced Hunting, Sentinel Graph, threat hunting notebooks | Domain 3 |
| **Week 6** | Full timed mock exams, gap review, lab reinforcement — book exam | All |

> ⏱️ Recommended pace: **10–12 hours/week**  
> 🎯 Book exam when MeasureUp mock consistently hits **80%+**

## ⚠️ Common Failure Points

- **Automation Rules ≠ Playbooks** — Automation rules are native Sentinel (no Logic Apps). Playbooks are Logic Apps. Confusing these costs marks.
- **Sentinel vs. Defender for Cloud alert origin** — Know which tool generates vs. aggregates alerts.
- **Skipping KQL lab practice** — Reading KQL is not enough. Write queries in a live Sentinel workspace.
- **Using pre-April 2026 materials** — The new objectives (Sentinel Graph, Copilot AI, Data Lake KQL) are not covered in older prep content.
